IF语句运行原理就是：给出条件，决定下一步怎么做？如果条件为真，就执行决策条件代码块的内容，为假就退出。

我们学习之前先看下Python中的真假：在python中，任何非零，非空对象都是真，除真和None以外其他的都是假。

来敲一下笔记：

  • 1.任何非零和非空对象都为真 解释为True
  • 2.数字0、空对象和特殊对象None均为假 解释为False
  • 3.比较和相等测试会应用到数据结构中
  • 4.返回值为True或False
